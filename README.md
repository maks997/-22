# Random Quote Generator

## Автор

## Описание

Приложение генерирует случайные цитаты, сохраняет историю и позволяет фильтровать их по автору и теме.

## Возможности

- Генерация случайной цитаты

- Сохранение истории

- Фильтрация по автору и теме

- Добавление новых цитат

- Сохранение данных в JSON

## Запуск

python main.py
